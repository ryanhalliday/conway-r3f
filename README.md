# Conway's Game of Life in React Three Fiber

## Playing

1. CLick on the grid to mark a cell as alive
2. Click "Start" to run the simulation
3. Watch it go!

## Running

```bash
npm install
npm run dev
```

## Exporting / Releasing

```bash
npm run export
# sync the out/ directory to a webserver
```
